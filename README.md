# Gemstone Grading and Generation Pipeline Code

This readme contains the code files, outputs, and plots generated for supporting our work on Gemstone Grading and Generation of Gemstone using CycleGAN pipeline. 
The repository is divided into the following :
- Notebooks contains the notebooks used for writing the code and generating the results.
- Outputs contains the plots, heatmaps, gemstone images received in the paper. The folder UV_Vis_NIR contains the plots and results obtained on using the UV-Vis_NIR plots from Gemology website.
- The UV-Vis-NIR plots were used for analysis for finding the gemstone location, color, and properties that the gemstone contained which was used for finding the Gemstone location. All of the CSV files in the repository contains the data generated from the UV-VIS-NIR plots for indentifying the location
of the gemstone.
- The images.zip file contain all of the emerald images used for gemstone grading and subsequent notebooks with plots of which have been provided in the repository for reprodcubility.

All the notebooks contain the plots and everything needed for reproducing our results.
